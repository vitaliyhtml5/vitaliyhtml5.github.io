Ключевое слово order by и limit

SELECT * FROM friends
ORDER BY name
LIMIT 3;

  • Order by - группирует а алфавитном порядке (от Я до А) и цифры по нарастающей
  • Можно поменять направление через слово DESC. Например, для порядка от А до Я можно прописать SELECT * FROM friends
    ORDER BY name DESC;
  • Limit - ограничивает число строк. На данном примере только 3 строки отображены

Соединение таблиц

Внутреннее соединение таблиц

У нас уже есть "главная" таблица friends. Создадим еще одну таблицу hobby, внешний ключ которой будет связан с первичным ключом таблицы friends.

CREATE TABLE hobby (
id int auto_increment not null PRIMARY KEY,
passion varchar(10),
id_friends int,
CONSTRAINT fk_friends_id
FOREIGN KEY(id_friends)
REFERENCES friends_id); Просмотреть структуру таблицы: DESC hobby;

id name age birth job
1 ross 32 1971-02-03 scientist

У нас две таблицы, которые связаны друг с другом первичным и внешним ключом.

id name age birth job
1 ross 32 1971-02-03 scientist
id name age birth id_friends
1 ross 32 1971-02-03 scientist

Выведем имена из таблицы friends и соединим с хобби из таблицы hobby.

SELECT friends.name, hobby.passion
FROM friends
INNER JOIN hobby
ON friends.id = hobby.id_friends;

id name age birth job
1 ross 32 1971-02-03 scientist

Внутреннее соединение с оператором where

SELECT friends.name, hobby.passion
FROM friends
INNER JOIN hobby
ON friends.id = hobby.id_friends;
WHERE friends.name = 'ross' AND friends.name = 'monica'

id name age birth job
1 ross 32 1971-02-03 scientist
1 ross 32 1971-02-03 scientist
