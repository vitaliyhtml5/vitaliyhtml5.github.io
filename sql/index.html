Создать таблицу и внести значения

Create table (без первичного ключа)

CREATE TABLE friends (
name varchar(10),
age int,
birth date
job varchar(12)
);

Просмотреть структуру таблицы: DESC friends;
name age birth job
Ross 32 1971-02-03 scientist
Ross 32 1971-02-03 scientist
Ross 32 1971-02-03 scientist

Create table (с первичным ключом)

CREATE TABLE friends (
id int auto_increment not null PRIMARY KEY,
name varchar(10),
age int,
birth date
job varchar(12)
);

Просмотреть структуру таблицы: DESC friends;
id name age birth job
1 ross 32 1971-02-03 scientist
2 ross 32 1971-02-03 scientist
3 ross 32 1971-02-03 scientist

Insert Into (одна строка)

INSERT INTO friends (name, age, birth, job)
VALUES ('ross', 32, '1971-02-03', 'scientist');

Просмотреть данные таблицы: SELECT * FROM friends;
id name age birth job
1 ross 32 1971-02-03 scientist

Insert Into (данные известны частично)

INSERT INTO friends (name, age, job)
VALUES ('rachel', 26 'waitress');

Просмотреть данные таблицы: SELECT * FROM friends;
id name age birth job
1 ross 32 1971-02-03 scientist
2 rachel 26 null waitress

Ключевое слово WHERE

SELECT * FROM friends
WHERE name = 'phibie';

id name age birth job
1 ross 32 1971-02-03 scientist

Оператор AND

SELECT * FROM friends
WHERE name = 'chandler'
AND name = 'monica';При AND все условия должны выполняться. Если бы мы вместо 'monica' написали 'jeniss', таблица не выдала бы нам никаких данных (даже по 'monica'). Так как два условия не были выполнены

id name age birth job
1 ross 32 1971-02-03 scientist
1 ross 32 1971-02-03 scientist

Оператор OR

SELECT * FROM friends
WHERE name = 'chandler'
OR age = 26;При OR должно выполниться хотя бы одно условие.

Вывод значений NULL

SELECT * FROM friends
WHERE age IS NULLЕсли нужны значения "не null", используем команду:
SELECT * FROM friends WHERE age IS NOT NULL;

Операторы сравнения

SELECT * FROM friends
WHERE age > 20Можно использовать следующие операторы:

  • Больше: >
  • Меньше: <
  • Больше или равно: > =
  • Меньше или равно: < =
  • Не равно: <>

id name age birth job
1 ross 32 1971-02-03 scientist
1 ross 32 1971-02-03 scientist
1 ross 32 1971-02-03 scientist

Оператор LIKE

SELECT * FROM friends
WHERE name LIKE 'Phi%' Несколько важных моментов:

  • Знак процента можно ставить в любой части слова
  • кавычки обязательны (даже для чисел)

id name age birth job
1 ross 32 1971-02-03 scientist

Ключевое слово ORDER BY и LIMIT

SELECT * FROM friends
ORDER BY name
LIMIT 3;

  • Group by - группирует а алфавитном порядке (от Я до А) и цифры по нарастающей
  • Можно поменять направление через слово DESC. Например, для порядка от А до Я можно прописать SELECT * FROM friends
    ORDER BY name DESC
  • Limit - ограничивает число строк. На данном примере только 3 строки отображены
